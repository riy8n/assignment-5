2. innerText is used for
  • find out any text from any tag
  • It cannot read any space or others

3. textContent is used for
  • Find out hidden text or set text
  • It cannot process html tag


Question-3: What is event delegation in the DOM?

Event delegation is a system which is used for added a single event listener instead of multiple listener. such as :

  • Instead of adding event listeners to multiple child elements, we add one event listener to the parent.
  • When an event occurs on a child, it "bubbles up" to the parent.
  • The parent checks which child triggered the event and handles it accordingly.


Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where an event starts from the target element and propagates upward to its ancestors like parent, grandparent, etc allowing event handlers on parent elements to execute as well.
